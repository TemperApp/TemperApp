# Temperapp :notes:



## Installation

```
npm install -g @ionic/cli
cd temperapp
ionic capacitor add android
ionic capacitor copy android
ionic capacitor run android -l --host=YOUR_IP_ADDRESS
(si marche pas ANDROID_SDK introuvable → ajouter “sdk.dir = C:/Users/ludch/AppData/Local/Android/Sdk” dans “./android/local.properties”
```

## Project Creation 

```
npm install -g @ionic/cli
ionic start temperapp blank
cd temperapp
ionic capacitor add android
ionic capacitor copy android
ionic capacitor run android -l --host=YOUR_IP_ADDRESS
(si marche pas ANDROID_SDK introuvable → ajouter “sdk.dir = C:/Users/ludch/AppData/Local/Android/Sdk” dans “./android/local.properties”
```
